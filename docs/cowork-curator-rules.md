# Curator Rules — Critérios de Curadoria

## Thresholds de Stale
- Ideia sem update há 30+ dias → flag como stale
- Ideia sem update há 60+ dias → sugerir arquivo
- Ideia com score BS < 2.0 → sugerir arquivo imediato

## Critérios de Merge
- 2 ideias com >60% overlap temático → candidatas a merge
- Merge preserva o item mais recente como winner
- Loser recebe tag `merged-into:{winner-id}` e stage → archived

## Completude pra PROMOTE
- [ ] Título PT + EN (ambos obrigatórios)
- [ ] Hook (1-2 frases, não genérico)
- [ ] Formato definido (video/blog/newsletter/social)
- [ ] Playlist target sugerida
- [ ] ≥2 tags relevantes
- [ ] Idioma(s) target

## Tags Obrigatórias
- Toda ideia: ≥1 playlist tag (a-g, s)
- Toda ideia: ≥1 formato tag (video, blog, newsletter, social, short)

## Limites
- Backlog max saudável: 30 ideias em stage=ideation
- Acima de 30: curadoria recomendada
- Acima de 50: curadoria urgente
